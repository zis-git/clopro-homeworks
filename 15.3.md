## Задание 1. Yandex Cloud

### Цель работы
С помощью ключа в Yandex Key Management Service (KMS) зашифровать содержимое ранее созданного bucket в Yandex Object Storage.

---

## Что было сделано

1. Использован ранее подготовленный bucket `dz15-3.uptime-ai.ru`.
2. Bucket был импортирован в Terraform state, так как он уже существовал:

```bash
terraform import yandex_storage_bucket.domain_bucket dz15-3.uptime-ai.ru
```

3. Создан симметричный ключ в Yandex KMS:
   - имя ключа: `dz15-3-bucket-key`;
   - алгоритм: `AES_128`;
   - период ротации: `8760h`.
4. Для bucket `dz15-3.uptime-ai.ru` включено шифрование объектов на стороне сервера через KMS.
5. Выполнена проверка итогового Terraform state для bucket и KMS-ключа.

---

## Terraform-манифесты

### `main.tf`

```hcl
terraform {
  required_providers {
    yandex = {
      source = "yandex-cloud/yandex"
    }
  }
}

provider "yandex" {
  folder_id = "b1g7jt8jrg491cftc7qa"
  zone      = "ru-central1-a"
}
```

### `storage-kms.tf`

```hcl
resource "yandex_kms_symmetric_key" "bucket_key" {
  name              = "dz15-3-bucket-key"
  description       = "KMS key for dz15-3.uptime-ai.ru bucket"
  default_algorithm = "AES_128"
  rotation_period   = "8760h"
}

resource "yandex_storage_bucket" "domain_bucket" {
  folder_id = "b1g7jt8jrg491cftc7qa"
  bucket    = "dz15-3.uptime-ai.ru"

  website {
    index_document = "index.html"
  }

  server_side_encryption_configuration {
    rule {
      apply_server_side_encryption_by_default {
        kms_master_key_id = yandex_kms_symmetric_key.bucket_key.id
        sse_algorithm     = "aws:kms"
      }
    }
  }
}
```

      description = "KMS key algorithm"
      type        = string
      default     = "AES_128"
    }

    variable "kms_key_rotation_period" {
      description = "KMS key rotation period"
      type        = string
      default     = "8760h"
    }

    variable "bucket_name" {
      description = "Domain bucket name"
      type        = string
      default     = "dz15-3.uptime-ai.ru"

    variable "website_index_document" {
      description = "Website index document"
      type        = string
      default     = "index.html"

    resource "yandex_kms_symmetric_key" "bucket_key" {
      name              = var.kms_key_name
      description       = var.kms_key_description
      default_algorithm = var.kms_key_algorithm
      rotation_period   = var.kms_key_rotation_period

      }
